# Module 1 — How software works (mental models)

Module 1 builds the durable mental models a learner needs before touching any tool. It is delivered as three bundled lessons (per D-06): client/server and browsers, databases and APIs, and authentication and deployment. Each bundle is anchored by a concrete analogy so a learner without prior coding experience has something to picture, not just terminology to memorize. Module 1 is intentionally pre-loop — every Loop check in this module names `intent` (per D-05) because the loop itself is taught in Module 3.

## Lessons in this module

- `01-how-the-web-works.md` — Client/server + browsers (restaurant analogy) *(Plan 01-4)*
- `02-where-data-lives.md` — Databases + APIs (filing cabinet + inter-office mail) — **exemplar lesson, ships in this plan**
- `03-who-can-do-what.md` — Auth + deployment (door staff + opening night) *(Plan 01-4)*
