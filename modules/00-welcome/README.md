# Module 0 — Welcome

Module 0 is the on-ramp. It welcomes the learner, walks through the hardware check (Codespaces vs local), then runs cost-path triage **before** any account is created (per D-16) so a learner only signs up for the AI-tool account their chosen path actually needs. The module ends with account creation and a Codespaces walkthrough so the learner is ready to enter Module 1 with a working environment and a clear cost path.

## Lessons in this module

- `01-welcome.md` — Welcome, who this is for, what to expect *(Plan 01-3)*
- `02-hardware-check.md` — Codespaces vs local; what your computer needs *(Plan 01-3)*
- `03-cost-path-triage.md` — Pick your cost path before creating accounts *(Plan 01-3)*
- `04-account-creation.md` — Create only the accounts your chosen path needs *(Plan 01-3)*
- `05-codespaces-walkthrough.md` — First-run walk through your Codespace *(Plan 01-3)*
