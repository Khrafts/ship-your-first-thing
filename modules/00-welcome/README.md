# Module 0 — Welcome

Module 0 prepares your environment so the rest of the course flows: welcome, hardware check, cost-path triage (BEFORE account creation per Phase 1 D-16), account creation, and a first-run walkthrough of GitHub Codespaces.

## Lessons in this module

1. [01-welcome.md](./01-welcome.md) — Welcome, who this is for, what to expect
2. [02-hardware-check.md](./02-hardware-check.md) — Codespaces vs local install; what your computer needs
3. [03-cost-path-triage.md](./03-cost-path-triage.md) — Pick your cost path before creating accounts
4. [04-account-creation.md](./04-account-creation.md) — Create only the accounts your chosen path needs
5. [05-codespaces-walkthrough.md](./05-codespaces-walkthrough.md) — First-run walk through your Codespace
